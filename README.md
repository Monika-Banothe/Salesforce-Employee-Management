# Salesforce Employee Management System

## Overview
This is a beginner Salesforce Admin project created for practice.

## Features
- Custom Object Creation
- Custom Fields
- Validation Rules
- Reports
- Dashboard

## Object Created
Employee

## Fields
- Employee ID
- Email
- Salary
- Department
- Joining Date

## Validation Rule
Salary cannot be Negative.
Email Address is a Mandatory.

## Report
Employee Report

## Dashboard
Employee by Department

## Tools Used
- Salesforce Playground
